# PPA Scoring for the secworks AES Core

To gather the scoring metrics for the AES Core please place a copy of this directory in the top-level of the secworks AES repo and run the `run_ppa.sh` script. In order for this script to work you must have installed [Yosys](https://yosyshq.net/yosys/) and [OpenSTA](https://github.com/The-OpenROAD-Project/OpenSTA).
